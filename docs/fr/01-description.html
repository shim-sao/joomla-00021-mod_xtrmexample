Introduction

Le Module présenté ici est un simple exemple de création d'un module pour Joomla. Il se compose pour la partie du site de l'affichage d'un message basique de bienvenue et pour la partie d'administration des paramètres de base servant à la configuration du module.
Nous allons voir par la suite en détail de quoi se compose le module ainsi que les différentes possibilités qu'offre le développement d'un module pour Joomla adapté à ses besoins.

Présentation

Tout d'abord, qu'est-ce-qu'un Module pour Joomla ?
Un module pour Joomla est une extension composées de fichiers dans Joomla qui permettent d'afficher des éléments préconfigurés dans les positions prédéfinies du template que l'on souhaite.
Comme exemple de module nous pouvons citer: le formulaire de connexion, la liste des articles les plus visités...
Les modules se trouve dans le répertoire /modules de Joomla.
