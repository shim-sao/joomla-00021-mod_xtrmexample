Introduction

Le Module présenté ici est un simple exemple de création d'un module pour Joomla. Il se compose, pour la partie du site, de l'affichage d'un message basique de bienvenue, et pour la partie d'administration, des paramètres de base servants à la configuration du module.
Nous allons voir par la suite en détail de quoi se compose le module ainsi que les différentes possibilités qu'offre le développement d'un module pour Joomla adapté à ses besoins.

Présentation

Tout d'abord, qu'est-ce-qu'un Module pour Joomla ?
Un module pour Joomla est une extension composées de fichiers dans Joomla qui permettent d'afficher des éléments préconfigurés dans les positions prédéfinies que l'on souhaite à l'intérieur du template.
Comme exemple de module nous pouvons citer: le formulaire de connexion, la liste des articles les plus visités...

Tous les modules se trouvent dans (ou commencent par) le répertoire /modules de Joomla.

Notez ici que la position ainsi que l'affichage dépendra du choix que vous aurait fait dans la configuration du votre module.
